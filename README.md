# Машинное обучение: Предсказание оттока клиентов телеком-компании

## Описание проекта

Этот проект демонстрирует полный цикл решения задачи машинного обучения - предсказание оттока клиентов (churn prediction) телекоммуникационной компании. Проект в рамках дисциплины "Основы машинного обучения с приложениями в финансах" и включает все этапы от анализа данных до развертывания модели.

### Цели проекта:
- Провести полный анализ данных (EDA)
- Построить и сравнить несколько моделей ML
- Выбрать лучшую модель для предсказания оттока
- Обеспечить интерпретируемость результатов
- Сохранить модель для использования в продакшене

## Данные проекта:

**Датасет**: Telco Customer Churn
- **Размер**: 7,043 строки × 21 столбец
- **Целевая переменная**: Churn (Yes/No) - бинарная классификация
- **Признаки**: демографические данные, информация об услугах, платежах, контрактах

## Что использовалось в рамках проекта:

- **pandas** - обработка данных
- **numpy** - числовые вычисления
- **matplotlib** + **seaborn** - визуализация
- **scikit-learn** - машинное обучение
- **joblib** - сохранение модели

### Ожидаемые метрики:
- **ROC-AUC**: 0.83-0.85 (на тестовой выборке)
- **Accuracy**: ~79%
- **Precision (Churn)**: ~63%
- **Recall (Churn)**: ~51%


## Модели и метрики:

### Используемые модели:
1. **LogisticRegression** - базовая модель, интерпретируемая
2. **RandomForest** - ансамблевая модель, устойчивая к переобучению

### Метрики оценки:
- **ROC-AUC** - основная метрика (для несбалансированных данных)
- **Precision/Recall/F1** - дополнительные метрики
- **Accuracy** - общая точность

## Ключевые выводы:

### Факторы оттока клиентов:
1. **tenure** (стаж клиента) - самый важный фактор
2. **MonthlyCharges** - высокие платежи коррелируют с оттоком
3. **Contract** - month-to-month контракты имеют высокий риск
4. **InternetService** - Fiber optic увеличивает риск оттока

### Рекомендации:
- Фокус на удержании долгосрочных клиентов
- Стимулирование перехода на годовые контракты
- Мониторинг клиентов с высокими месячными платежами
- Предложение специальных условий для fiber optic клиентов
